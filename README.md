# 📚 My Study Diary

> 🌐 **배포 사이트**  
> https://mystudydiary.netlify.app/



하루 학습을 기록하고 돌아볼 수 있는 웹 기반 학습 다이어리입니다.

---

## 📖 프로젝트 소개

**My Study Diary**는 하루 동안 학습한 내용을 기록하고,
목표와 어려웠던 점, 새롭게 배운 내용을 정리할 수 있는
개인 학습 기록 웹 애플리케이션입니다.

브라우저의 **LocalStorage**를 활용하여
별도의 서버 없이도 학습 기록을 저장하고 관리할 수 있으며,
**JSON 백업 및 복원 기능**을 통해 데이터를 안전하게 보관할 수 있습니다.

---

## ✨ 주요 기능

- 📝 학습 기록 작성
- ✏️ 기록 수정
- 🗑️ 기록 삭제
- 📅 같은 날짜에 여러 개의 학습 기록 저장
- 🗂️ 날짜별 기록 그룹화
- 🔍 날짜 검색
- 🔎 제목, 학습 주제, 내용 키워드 검색
- 💡 검색 결과 미리보기 및 검색어 강조
- 📖 학습 기록 상세 보기
- 🌿 작성 중 자동 임시 저장 및 복원
- 🗓️ 사이드바 달력 제공
- 📍 기록이 있는 날짜 표시
- 📤 JSON 형식으로 기록 내보내기
- 📥 JSON 형식으로 기록 가져오기
- 📱 모바일 환경을 고려한 반응형 UI

---

## 💾 데이터 저장 방식

학습 기록은 브라우저의 **LocalStorage**에 저장됩니다.

데이터를 안전하게 관리할 수 있도록 다음 기능을 제공합니다.

- 📤 JSON 파일로 기록 내보내기
- 📥 JSON 파일에서 기록 가져오기

## 📌 프로젝트를 통해 배운 점

- LocalStorage를 활용한 데이터 저장 및 관리 방법을 익혔습니다.
- JavaScript를 이용하여 CRUD(생성, 조회, 수정, 삭제) 기능을 구현했습니다.

- 반응형 웹 디자인을 적용하여 모바일 환경에서도 사용할 수 있도록 구현했습니다.
- GitHub와 Netlify를 활용한 프로젝트 배포 과정을 경험했습니다.
